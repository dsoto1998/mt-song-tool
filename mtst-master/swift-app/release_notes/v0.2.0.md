# MT Song Tool — v0.2.0

## Features

- Parse Ableton Live (.als) files to extract locators and time signature changes
- Drag-and-drop or click-to-browse file loading
- Copy individual timecodes or copy all at once
- Locator label validation — labels checked against accepted MultiTracks section list; invalid labels highlighted in red
- **Song Data panel** — Collects song metadata after parsing:
  - Song Key (dropdown with 24 approved keys)
  - Time Signature (auto-populated from first time sig, dropdown with 19 approved values)
  - BPM (auto-populated from Ableton session tempo)
  - Preview Start (auto-calculated: first Chorus after 01:00 minus 10 seconds, in whole seconds)
  - Preview End (auto-calculated: Preview Start + 45 seconds)
  - RehearsalMix Only (checkbox)
- Send to Song Shell (MTID) input placeholder
- Native macOS app with transparent title bar and traffic lights
- Custom MTST branding with Horizon font
- Portable — no Python or dependencies needed on target Mac
- .pkg installer for easy distribution
