# MT Song Tool — v0.1.4

## Features

- Parse Ableton Live (.als) files to extract locators and time signature changes
- Drag-and-drop or click-to-browse file loading
- Copy individual timecodes or copy all at once
- **Locator label validation** — labels are checked against the accepted MultiTracks section list; invalid labels highlighted in red
- Native macOS app with transparent title bar and traffic lights
- Custom MTST branding with Horizon font
- Portable — no Python or dependencies needed on target Mac
- .pkg installer for easy distribution

## Changelog

### v0.1.4 — March 12, 2026
- **Locator label validation** — Locator labels are validated against the official MultiTracks song sections list. Labels must be ALL CAPS, no extra spaces, and match a recognized section name. Invalid labels are highlighted in red (text, row number, timecode, and background) so formatting errors are immediately visible
- **MTID input** — Added MTID number input field in the results toolbar (placeholder — full functionality coming soon)

### v0.1.3 — March 12, 2026
- **Copy button hover glow** — Copy icons now highlight blue with a subtle background glow on hover, making it clear which row you're about to copy
- **Unified font** — Timecodes now use the same font as marker labels for a cleaner, easier-to-read look
- **Row numbers** — Each locator and time signature now shows its index number for easy reference

### v0.1.2 — March 12, 2026
- **Faster parsing** — Parser now pre-warms on app launch, so .als files parse nearly instantly
- **Faster startup** — Switched from single-file to directory-based bundling, eliminating extraction delay
- **Versioned releases** — Installer packages now include version labels and release notes

### v0.1.1 — March 12, 2026
- Rebuilt front-end in Swift/SwiftUI for a native macOS experience
- Transparent title bar with native traffic lights (no more ctypes crash)
- Bundled parser as standalone binary — fully portable, no Python needed on target Mac

### v0.1.0 — March 12, 2026
- Initial release — PySide6 (Python) UI with full parsing functionality
